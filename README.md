# Personal HTML Projects

## Overview

This repository contains multiple personal web development projects built using HTML.  

The projects include:

- A personal resume website
- A blog website

These projects demonstrate foundational front-end development skills, clean structure, and attention to layout and design.

## Technologies Used

- HTML5
- CSS3
- Responsive Web Design principles
- Git & GitHub
- Visual Studio Code

## Projects Included

### Resume Website
A personal resume website designed to showcase:
- Professional experience
- Technical skills
- Education
- Contact information
- Clean and responsive layout

### Blog Website
A simple blog website designed to demonstrate:
- Page structure
- Content organization
- Navigation
- Multi-page layout
- Readable formatting

## Project Structure

Each project contains:
- index.html
- CSS files (if applicable)
- Images/assets folder (if used)

## Purpose of These Projects

These projects were created to:
- Strengthen front-end development fundamentals
- Practice semantic HTML structure
- Improve layout organization
- Build a professional web portfolio
- Demonstrate Git version control usage

## Future Improvements

- Add responsive design enhancements
- Improve styling with advanced CSS
- Add JavaScript interactivity
- Optimize accessibility
- Deploy live versions using GitHub Pages

## How to View

1. Clone the repository
2. Open any `index.html` file in a browser
3. Or deploy using GitHub Pages
